P5.js on Oscilloscopes

Lecturer

Ted Davis

Team

Ayleen Weidmann

In this experimental project, we coded various Mandala-inspired shapes and allowed them to interact with each other through different signals. Using a MIDI controller, we were able to manipulate values and experiment with shapes, speed, and intersections. As the medium, we used an oscilloscope to display the code directly on the instrument.
